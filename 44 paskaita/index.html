Git ir GitHub

Git'as - kodo versijavimo programa; GitHub - versijuoto kodo talpykla

Git komandos

  • init - sukurti repozitoriją
  • status - pateikia git repozitorijos dabartinę informaciją
  • add - prideda nurodytus failus prie commitinimui paruoštų failų
  • rm --cached || restore --staged - išima nurodytus failus iš stage zonos
  • commit -m "Žinutė apie commit'ą" - sukuria kodo versiją iš stage failų ir prideda žinutę apie pakitimus
  • restore - sugrąžina nurodytą/us failą/us į pradinę būseną (jeigu tie failai nebuvo add'imti į stage'ingo zoną)
  • push
  • pull
